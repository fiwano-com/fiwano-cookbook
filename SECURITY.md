# Security Policy

## Reporting a vulnerability

If you discover a security issue — in these examples or in the Fiwano platform — please **do not open a public issue**. Email **contact@fiwano.com** with the details and we will respond as quickly as we can.

Please include, where possible:

- a description of the issue and its impact,
- steps to reproduce,
- any relevant logs or proof-of-concept (with secrets redacted).

## Scope

This repository contains example code and a generated mirror of our public documentation. The Fiwano platform itself is closed-source; vulnerability reports about the live API are equally welcome at the address above.

## Please never share secrets

Do not post API keys, access tokens, or webhook secrets in issues, discussions, or commits. If you accidentally expose a key, rotate it from your Fiwano dashboard immediately.
